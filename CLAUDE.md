# Klymb Website — Claude Code Instructions

## Project
Single-file website: `index.html`
Live at: klymb-website.vercel.app (auto-deploys on push to main)
GitHub: https://github.com/emfpublishing/klymb-website

## Workflow
1. Edit `index.html` directly
2. `git add index.html && git commit -m "<description>" && git push`
3. Vercel auto-deploys within ~30 seconds

## Rules
- All edits go in `index.html` — no other files needed
- Always push after every edit
- Keep the single-file structure — no build tools, no dependencies
